Tournament Tracker for golf

Capstone Project built at Actualize

Description

Coming from a family of golfers that participate in annual tournaments, I wanted to create an application that could be used to seamlessly plan and track all the rounds and scores that take place every year.

So I built this web application, called Tournament Tracker, that allows golf players to do just that.

I built the API using Ruby on Rails and the front end with Vue.js, using a bootstrap theme to incorporate modern visual stylings and intuitive site navigation.. Commissioners can create a tournament and manage match schedules and scoring, while all players can log their individual scores and see scores update live.
